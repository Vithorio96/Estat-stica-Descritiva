# Estatistica Descritiva
Análise de Estatística Descritiva sobre a Produção de Leite (Região Sul) utilizando dados do IBGE/SIDRA.

Este repositório contém uma análise prática de Estatística Descritiva aplicada a dados reais do setor agropecuário brasileiro. O objetivo principal é consolidar conceitos fundamentais de análise exploratória de dados (EDA).

Conteúdo explorado:

Medidas de Posição: Média, mediana e moda para entender o comportamento central da produção.

Medidas de Dispersão: Variância, desvio padrão e coeficiente de variação para avaliar a volatilidade e consistência dos dados.

Propriedades de Distribuição: Análise de simetria e curtose para identificar o perfil da distribuição da produção de leite.

Dados: Os dados foram extraídos da Pesquisa Trimestral do Leite (IBGE/SIDRA), com foco exclusivo na região Sul do Brasil.
